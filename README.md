# Project_Foundation_of_AI## Visual Features

- The program supports **colored output** using `colorama`:
  - `#` walls are shown as blocks `█`
  - `S` (start) in green
  - `G` (goal) in red
  - Path cells as yellow `•`
  - Explored cells as blue `·`

To install `colorama` (optional):

```bash
pip install colorama
